# PWA Icons

Placeholder SVG icon created. 

## Production Icons

For production, generate proper PWA icons from `icon.svg`:

```bash
# Using PWA Asset Generator (recommended)
npx pwa-asset-generator icon.svg ./icons --manifest ../manifest.json

# Or use https://realfavicongenerator.net/
# Upload icon.svg and generate all required sizes
```

## Required Sizes

- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Apple Touch Icons

Additional sizes for iOS:
